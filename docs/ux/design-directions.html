Design Notes: VM Request Form

  • Multi-Step Form: Breaks complex request into digestible steps (Basics → Resources → Review)
  • VM Size Selector: Visual card selection instead of confusing dropdowns - "S/M/L/XL" is intuitive
  • Quota Indicator: Real-time feedback on remaining resources - prevents failed requests
  • Clear Labels: Required fields marked, helpful placeholder text, validation hints
  • Consumer Simplicity: Looks like ordering something online, not filling out enterprise forms
