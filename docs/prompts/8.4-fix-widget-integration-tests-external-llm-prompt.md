### PostgreSQL Behavior
- Unquoted identifiers: Converted to lowercase (`widget_projection`)
- Quoted identifiers: Case-preserved (`"WIDGET_PROJECTION"`)
- Current DDL creates: `CREATE TABLE widget_projection` (lowercase)
- jOOQ queries: `SELECT ... FROM "WIDGET_PROJECTION"` (uppercase, quoted)
- Result: Mismatch → "relation does not exist"
