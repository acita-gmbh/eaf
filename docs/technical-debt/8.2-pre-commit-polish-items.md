### 2. Add CI Hook Validation Job

**Scope**: Create CI job to ensure hook config matches CI rules (AC17)

**Work Required** (1 day, Priority: P3):

**File**: `.github/workflows/validate-hooks.yml`

**Job Steps**:
1. Check `.pre-commit-config.yml` exists
2. Verify hook rules subset of CI rules
3. Run `./gradlew installGitHooks` in CI
4. Execute hooks in CI environment
5. Fail if hooks don't match CI behavior

**Value**: Prevents drift between local hooks and CI validation
