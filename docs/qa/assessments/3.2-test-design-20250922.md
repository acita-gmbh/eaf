## Detailed Test Scenarios

### Priority 0 (Critical) Tests

#### 3.2-UNIT-003: JWT Decoder Bean Creation
**AC**: AC2 - OIDC Discovery Configuration
**Test**: Verify JwtDecoder bean is created with correct OIDC issuer URL
**Given**: Spring Security configuration class
**When**: Application context loads
**Then**: JwtDecoder bean should be available with Keycloak discovery endpoint
**Mitigates**: TECH-001

#### 3.2-UNIT-004: RS256 Algorithm Enforcement
**AC**: AC2 - Enhanced Security (Subtask 2.5)
**Test**: Verify JWT decoder only accepts RS256 algorithm
**Given**: JWT decoder configuration
**When**: Token with non-RS256 algorithm is processed
**Then**: Token should be rejected with algorithm validation error
**Mitigates**: SEC-001

#### 3.2-INT-008: Unauthenticated Request Security Boundary
**AC**: AC4 - Integration Testing
**Test**: Verify secured endpoint rejects requests without authentication
**Given**: Secured endpoint `/api/secure/hello`
**When**: Request made without Authorization header
**Then**: Should return 401 Unauthorized with proper error message
**Mitigates**: Core security requirement

#### 3.2-INT-009: Valid Token Authentication Flow
**AC**: AC4 - Integration Testing
**Test**: Verify secured endpoint accepts valid JWT tokens
**Given**: Valid JWT token from Keycloak Testcontainer
**When**: Request made with valid Authorization header
**Then**: Should return 200 OK with JWT claims information
**Mitigates**: Core authentication functionality

#### 3.2-INT-010: Malformed JWT Attack Protection
**AC**: AC4 - Enhanced Security (Subtask 4.5)
**Test**: Verify robust handling of malformed JWT tokens
**Given**: Various malformed JWT tokens (invalid signature, structure)
**When**: Request made with malformed token
**Then**: Should return 401 Unauthorized without information leakage
**Mitigates**: SEC-001, TEST-001

#### 3.2-INT-011: Algorithm Confusion Attack Protection
**AC**: AC4 - Enhanced Security (Subtask 4.5)
**Test**: Verify protection against algorithm confusion attacks
**Given**: Tokens with HMAC, none, or other algorithms
**When**: Request made with non-RS256 algorithm token
**Then**: Should return 401 Unauthorized with algorithm validation error
**Mitigates**: SEC-001

#### 3.2-INT-012: Expired Token Handling
**AC**: AC4 - Enhanced Security (Subtask 4.6)
**Test**: Verify proper handling of expired tokens
**Given**: Expired JWT tokens with various expiration scenarios
**When**: Request made with expired token
**Then**: Should return 401 Unauthorized with expiration error
**Mitigates**: Time-based security

#### 3.2-INT-013: Issuer/Audience Validation
**AC**: AC4 - Enhanced Security (Subtask 4.7)
**Test**: Verify trusted source validation
**Given**: Tokens with invalid issuer or audience claims
**When**: Request made with untrusted token
**Then**: Should return 401 Unauthorized with validation error
**Mitigates**: SEC-002

## Test Implementation Guidelines

### Framework Standards
- **Kotest BehaviorSpec**: Use Given-When-Then structure for readable security scenarios
- **Spring Boot Test**: Use @SpringBootTest for integration tests
- **MockMvc**: Use for controller-specific security testing
- **TestRestTemplate**: Use for full HTTP request/response validation

### Security Test Patterns
```kotlin
// Example: Algorithm validation test
Given("a JWT decoder with RS256 enforcement") {
    When("processing token with HMAC algorithm") {
        val hmacToken = createHMACToken()
        val result = callSecuredEndpoint(hmacToken)

        Then("should reject with algorithm error") {
            result.statusCode shouldBe HttpStatus.UNAUTHORIZED
            result.body shouldContain "Unsupported algorithm"
        }
    }
}
```

### Integration Test Patterns
```kotlin
// Example: OIDC discovery integration
context("OIDC Discovery Integration") {
    test("should auto-configure with Keycloak Testcontainer") {
        // Given: Keycloak container running with EAF realm
        val discoveryUrl = "${keycloakContainer.authServerUrl}/realms/eaf/.well-known/openid_configuration"

        // When: Application starts
        val jwtDecoder = applicationContext.getBean(JwtDecoder::class.java)

        // Then: Should be configured with discovery endpoint
        jwtDecoder.shouldNotBeNull()
        // Verify OIDC configuration loaded correctly
    }
}
```
