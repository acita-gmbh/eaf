   - Requires Docker daemon running
   - Requires internet access for first-run container pulls
   - Requires sufficient system resources (4GB+ RAM recommended)
   - Developer onboarding complexity: `./scripts/init-dev.sh` prerequisite

4. **Missing Test Strategy**:
   - No guidance on when to run integration tests (pre-commit? CI only?)

   - No test suite prioritization for local development
