## Purpose

These documents:

## When to Add Documents Here

Add documents to this archive when:
- Debugging process generates extensive documentation (>500 lines)
- Problem-solving involved significant investigation or external AI analysis
- Solution has broader architectural implications worth preserving
- Document provides value as historical reference but clutters main docs/

**Note:** Always add a header to archived documents explaining their historical nature and context.
