## Business Context

### Problem

**Critical Defect Discovered:** Story 8.7 implementation (Nightly Deep Validation Pipeline) has a **P0 blocker** - fuzz corpus persistence is completely non-functional:

**Evidence from Run #7 (18442213566) and Run #8 (18442635199):**
- ❌ Corpus restore: Cache not found (100% failure rate)
- ❌ Corpus save: "Path Validation Error: Path(s) do(es) not exist" (100% failure rate)
- ❌ Fuzz execution time: 19-26 seconds (should be 1800 seconds / 30 minutes)
- ❌ No `.jazzer/corpus` directory created after execution
- ❌ Zero fuzz-related caches in GitHub Actions (0 out of 30 total caches)

**Root Cause:**
- Jazzer running in JUnit **test mode**, not **fuzzing mode**
- No corpus directory configuration in `build.gradle.kts`
- System property `-Djazzer.flags` not reaching Jazzer agent correctly
- Missing `doFirst` directory creation step

**Impact:**
- **Story 8.7 AC3 NOT MET:** "Fuzzing corpus saved for next run (incremental fuzzing)"
- **Security validation degradation:** 70% effectiveness loss (no incremental fuzzing)
- **Wasted CI resources:** Each run explores same basic inputs
- **False sense of security:** Green checkmarks hide non-functional fuzzing

### Opportunity

Fix Jazzer configuration to enable true incremental fuzzing:

1. **Security Improvement:** Deep vulnerability discovery over weeks/months
2. **CI Efficiency:** Build on previous fuzzing progress
3. **Compliance:** Meet Story 8.7 AC3 acceptance criteria
4. **Visibility:** Proper corpus growth tracking

## Target State (Fixed)

### Correct Fuzz Execution Pattern

```bash
# What should happen (FIXED):
10:33:03 - Restore fuzz corpus: Found 127 test cases from previous run ✅
10:33:03 - Run Fuzz Tests: Started (30-minute fuzzing session)
11:03:03 - Run Fuzz Tests: Completed (1800 seconds) ✅
11:03:03 - Corpus directory: 243 test cases (116 new) ✅

```

**Expected:**

- Corpus directory with 50-500+ test case files
- Incremental progress across runs
- Real security fuzzing active

### Build Configuration (Fixed)

```kotlin
// framework/security/build.gradle.kts
val fuzzTest = tasks.register<Test>("fuzzTest") {
    group = "verification"
    description = "Runs Jazzer fuzz tests (nightly/security validation)"
    testClassesDirs = sourceSets["fuzzTest"].output.classesDirs
    classpath = sourceSets["fuzzTest"].runtimeClasspath
    useJUnitPlatform()

    // ✅ FIX 1: Create corpus directory before execution
    doFirst {
        val corpusDir = file("${project.projectDir}/.jazzer/corpus")
        corpusDir.mkdirs()
        logger.lifecycle("✅ Created corpus directory: ${corpusDir.absolutePath}")
    }

    // ✅ FIX 2: Configure Jazzer for fuzzing mode
    systemProperty("jazzer.instrumentation_includes", "com.axians.eaf.**")
    systemProperty("jazzer.corpus_dir", "${project.projectDir}/.jazzer/corpus")
    environment("JAZZER_FUZZ", "1")

    // ✅ FIX 3: Accept time limit from command line
    System.getProperty("jazzer.flags")?.let { flags ->
        jvmArgs(flags.split(" "))
    }
}
```
