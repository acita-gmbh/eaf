## Top 5 Implementation Risks & Mitigations

### Risk 1: JWT localStorage XSS Vulnerability (Story 7.4a)
- **Score**: 6 (HIGH)
- **Mitigation**: Short-lived tokens (15min), DOMPurify, backend CSP headers, README advisory
- **Tests**: 7.4a-UNIT-P0-001 (XSS), P0-005 (expiration), P0-006 (refresh)

### Risk 2: Tenant Context Propagation Failure (Story 7.4a)
- **Score**: 6 (HIGH)
- **Mitigation**: Fail-closed validation (reject if missing tenant_id), backend 3-layer enforcement
- **Tests**: 7.4a-UNIT-P0-003 (tenant propagation)

### Risk 3: Story 7.4a Architectural Dependency (Story 7.4b)
- **Score**: 6 (HIGH)
- **Mitigation**: Sequential implementation (7.4a first), Task 3.6 validates framework exists
- **Tests**: 7.4b-UNIT-006 (framework existence), 7.4b-INTEGRATION-001 (import resolution)

### Risk 4: Path Traversal Attacks (Story 7.4b)
- **Score**: 6 (HIGH)
- **Mitigation**: 3-layer defense from Story 7.3 (input validation, sanitization, canonical path)
- **Tests**: 7.4b-UNIT-001 (attack scenarios - IMPLEMENT FIRST)

### Risk 5: Keycloak OIDC Implementation Errors (Story 7.4a)
- **Score**: 6 (HIGH)
- **Mitigation**: Comprehensive testing (6 tests), security architect review, follow Story 3.3 patterns
- **Tests**: 7.4a-UNIT-P0-004, P0-005, P0-006, P1-004, P1-005
